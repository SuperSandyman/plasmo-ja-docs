# HTMLテンプレートの置換

デフォルトのHTMLテンプレートを置き換えるには、コンポーネントファイルと同じ命名規則の`.html`ファイルを作成します。たとえば、`popup.tsx`があり、HTMLをオーバーライドしたい場合は、`popup.html`ファイルを作成します。

```html filename="popup.html"
<!DOCTYPE html>
<html>
  <head>
    <title>__plasmo_static_index_title__</title>
    <meta charset="utf-8" />
  </head>
  <body></body>
</html>
```

`popup/index.tsx`ファイルがある場合は、上記と同様の内容の`popup/index.html`ファイルを作成します。

<Callout emoji="👀">
ルート要素やscriptタグを含める必要はありません。Plasmoフレームワーク(Framework)が処理します。
</Callout>
